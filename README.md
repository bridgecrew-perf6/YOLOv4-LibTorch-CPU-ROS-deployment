# ROS-YOLOv4-LibTorch
this is a Deployment of YOLOv4  on ROS-Melodic with LibTorch

Environment： 
CMake 3.13.2 
OpenCV 3.4.10 
opencv-contrib-3.4.10 
torchvision-0.6.0 
LibTorch1.5.0-CPU
ROS-Melodic
  
this code is based on CPU，and you can refer the version based on GPU at https://github.com/wsx000/YOLO4-ROS-GPU
